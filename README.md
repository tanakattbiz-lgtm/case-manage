# 案件管理システム (Case Management System)

Google Apps Script (GAS) と スプレッドシートを活用した、シンプルかつ高機能な案件・クライアント管理システムです。
PC・モバイルの両方で快適に操作できるよう、レスポンシブなモダンUIを採用しています。

## 🚀 主な機能

### 1. ダッシュボード
- **KPI表示**: 累計売上、累計利益、進行中案件の売上、平均利益率などをリアルタイムに集計。
- **月次推移グラフ**: 過去12ヶ月の売上・利益の推移を視覚的に把握。
- **クライアントランキング**: 売上上位のクライアントをバーグラフで表示。
- **最新案件**: 直近で登録・更新された案件をクイック確認。

### 2. 案件管理
- **ステータス管理**: 商談中、進行中、完了、停止、保留の5段階で管理。
- **収益管理**: 売上・利益の入力。利益率の自動計算とカラー表示。
- **日付手入力**: 完了日や予定日を自由に設定可能。
- **柔軟なフィルタリング**: ステータスやクライアント、フリーワードによる高速検索。

### 3. クライアント管理
- **マスタ管理**: クライアント名、担当者、連絡先情報を整理。
- **案件紐付け**: 案件登録時に既存クライアントから選択、またはその場で新規登録が可能。
- **実績確認**: 各クライアントごとの累計売上や取引件数を表示。

## 🛠 プロジェクト構成

```
case-manage/
├── src/
│   ├── main.gs      # Google Apps Script バックエンドロジック（データ操作・ルーティング）
│   └── index.html   # フロントエンド（HTML/CSS/JS 全てを含むSPA）
└── スプレッドシート   # データベースとして「案件データ」「クライアントマスタ」シートを使用
```

## 📦 セットアップ方法

1.  **スプレッドシートの準備**
    - 新規の Google スプレッドシートを作成します。
2.  **スクリプトエディタの起動**
    - [拡張機能] > [Apps Script] を開きます。
3.  **コードの配置**
    - `src/main.gs` の内容を `コード.gs` に貼り付けます。
    - `src/index.html` を `index.html` という名前のHTMLファイルとして作成し、内容を貼り付けます。
4.  **デプロイ**
    - [デプロイ] > [新しいデプロイ] を選択します。
    - 種類を「ウェブアプリ」にし、アクセスできるユーザーを適切に設定してデプロイします。
5.  **シートの自動生成**
    - 初回実行時に「案件データ」「クライアントマスタ」の各シートとヘッダー行が自動的に生成されます。

## 🎨 デザインテーマ
- **UI**: Noto Sans JP を使用したクリーンなタイポグラフィ。
- **色設計**: 状態に応じたセマンティックカラー（進行中: 緑, 完了: 青, 警告: 赤 など）。
- **アニメーション**: モーダルのスライドアップ、ビューのフェードイン等による滑らかな操作感。
